1）组件设计：严格遵循单一职责原则，组件具有良好的可复用性和可维护性

## 设计思维
在编码之前，先了解上下文，并确定一个大胆（BOLD）的美学方向：
- **目的**：该接口解决了什么问题？谁在使用它？
- **色调**：选择一种极端风格：极简主义、极繁主义、复古未来主义、有机/自然、奢华/精致、俏皮/玩具风、编辑/杂志、野兽派/原始、装饰艺术/几何、柔和/粉彩、工业/实用等。有如此多的风格可供选择。利用这些风格作为灵感，但设计一个忠于美学方向的作品。
- **约束条件**：技术要求（框架、性能、可访问性）。
- **差异化**：是什么让这个体验令人难忘？有什么独特之处会让人铭记在心？
**关键提示**：选择一个清晰的概念方向，并精准执行。大胆的极繁主义和精致的极简主义都可行——关键在于意图，而非强度。
然后实现可运行的代码（HTML/CSS/JS、React、Vue等），该代码：
- 生产级且功能齐全
- 视觉上引人注目且令人难忘
- 连贯一致，具有明确的美学观点
- 每一处细节都经过精心打磨
## 前端美学指南
关注：
- **排版设计**：选择美观、独特且有趣的字体。避免使用Arial和Inter等通用字体；而应选择能提升前端美观度的独特字体；出人意料的、有特色的字体选择。将独特的显示字体与精致的正文字体搭配使用。
- **颜色与主题**：保持整体风格一致。使用CSS变量确保一致性。主色调搭配鲜明点缀，优于色彩柔和、分布均匀的调色板。
- **动态效果**：使用动画来增强效果和微交互。对于HTML，优先考虑仅使用CSS的解决方案。如果可用，请使用vue的antdesign库。关注高影响力的时刻：一个精心编排的页面加载，通过交错展示（动画延迟）比分散的微交互更能带来愉悦感。使用滚动触发和悬停状态来创造惊喜。
- **空间构成**：意想不到的布局。不对称。重叠。对角线流动。打破网格的元素。宽敞的负空间或受控的密度。
- **背景与视觉细节**：营造氛围和深度，而非仅使用纯色。添加与整体美学相匹配的情境效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。
切勿使用通用的人工智能生成的美学元素，如过度使用的字体系列（如Inter、Roboto、Arial、系统字体）、陈旧的配色方案（尤其是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏特定情境特征的千篇一律的设计。
进行创造性解读，并做出出人意料的、真正符合情境的选择。没有一种设计应该是千篇一律的。在明暗主题、不同字体、不同美学之间进行变化。绝不能在几代人之间采用共同的选择（例如Space Grotesk字体）。
**重要提示**：使实现复杂度与美学愿景相匹配。极繁主义设计需要精心编写的代码，并融入丰富的动画和效果。极简主义或精致设计则需要克制、精确，以及对间距、排版和细微细节的细致关注。优雅源于对愿景的出色执行。
记住：你有能力创作出非凡的作品。不要拘束，展现当跳出常规思维框架，并全身心投入于一个独特的愿景时，真正能够创造出的东西。

## 参考配置

1）vite.config.js 必须配置 base 路径以支持子路径部署、需要支持通过 @ 引入文件、不要配置端口号


import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  base: './',
  plugins: [vue()],
  resolve: {
    alias: {
      '@': fileURLToPath(new URL('./src', import.meta.url))
    }
  }
})


2）路由配置必须使用 hash 模式，避免服务器端路由配置问题

import { createRouter, createWebHashHistory } from 'vue-router'

const router = createRouter({
  history: createWebHashHistory(),
  routes: [
    // 路由配置
  ]
})


3）package.json 文件参考：

{
  "scripts": {
    "dev": "vite",
    "build": "vite build"
  },
  "dependencies": {
    "vue": "^3.3.4",
    "vue-router": "^4.2.4"
  },
  "devDependencies": {
    "@vitejs/plugin-vue": "^4.2.3",
    "vite": "^4.4.5"
  }
}
